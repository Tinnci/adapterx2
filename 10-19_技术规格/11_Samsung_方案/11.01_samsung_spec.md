# Samsung S9 专用 UART 转接板

## 设计代号

Project-S

## 电气规格

| 参数 | 值 | 说明 |
|------|-----|------|
| UART 电平 | 1.8V | ⚠️ 严禁使用 3.3V/5V |
| 激活电阻 | 619kΩ ±1% | B12 引脚对 GND |
| 信号引脚 | D+/D- | A6=RX, A7=TX |
| 波特率 | 115200 / 921600 | 可调 |

## MUIC 芯片信息

- 芯片型号: max77705 / s2mu004
- 驱动路径: `/sys/devices/virtual/sec/switch`
- 激活机制: 电阻 ID 检测 (RID)

## 关键约束

1. **不支持 USB 数据透传** - MUIC 会切断 D+/D-
2. **必须使用电平转换器** - 1.8V ↔ 3.3V
3. **右侧 Type-C 口仅供电** - 不要连接 USB 主机

## 内核配置 (可选)

```
CONFIG_CMDLINE="earlycon=exynos,0x13820000,115200n8 console=ttySAC2,115200"
CONFIG_CMDLINE_EXTEND=y
```

## 原理图要点

```
[手机端 Type-C 公头]                  [电平转换器]                 [PC端/排针]
      D+ (A6)  ---------------------> | LV_RX  HV_RX | ----------> RX (3.3V)
      D- (A7)  ---------------------> | LV_TX  HV_TX | ----------> TX (3.3V)
      B12      ------[619k Res]-----> GND
                                      |              |
      VBUS     -----+------------LDO->| 1.8V   3.3V  |<----------- 3.3V Input
                    |                 |______________|
                    |
                    +--------------------------------------------> [充电端 Type-C 母座 VBUS]
      GND      --------------------------------------------------> [充电端 Type-C 母座 GND]
```
