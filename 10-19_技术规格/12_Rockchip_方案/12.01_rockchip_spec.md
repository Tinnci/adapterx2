# Rockchip/PineNote 专用 UART 转接板

## 设计代号

Project-R

## 电气规格

| 参数 | 值 | 说明 |
|------|-----|------|
| UART 电平 | 3.3V | 标准 TTL 电平 |
| 激活电阻 | 2× 1kΩ | CC1/CC2 上拉至 3.3V |
| 信号引脚 | SBU1/SBU2 | A8=TX, B8=RX |
| 波特率 | 1,500,000 (1.5M) | 高速 |

## USB-C Debug Accessory Mode (DAM)

这是 USB Type-C 规范定义的标准调试模式。

当 CC 引脚检测到特定的上拉电阻时，USB 控制器会将 SBU1/SBU2 引脚直通到 SoC 的 UART 模块。

## 主要优势

1. ✅ **支持 USB 数据透传** - 可同时调试和传输文件
2. ✅ **无需电平转换** - 直接 3.3V
3. ✅ **标准协议** - 遵循 USB-C 规范

## 注意事项

- SBU 引脚可能根据线缆正反插变化，如无信号请尝试翻面
- 强行上拉 CC 会影响 PD 协议握手，建议加开关

## 原理图要点

```
[设备端 Type-C 公头]                                            [PC端 Type-C 母座]
      SBU1 (A8) -------------------------------------------------> (断开，不连母座)
         |
         +------------------------> UART TX (3.3V)
      
      SBU2 (B8) -------------------------------------------------> (断开，不连母座)
         |
         +------------------------> UART RX (3.3V)

      CC1/CC2   -------[1k Res]---< 3.3V (来自LDO或TTL工具)
                                     ^
                                     | (加个开关最好)
      
      D+/D-     -------------------------------------------------> D+/D- (数据直通)
      VBUS/GND  -------------------------------------------------> VBUS/GND (供电直通)
```

## 终端连接命令

```bash
minicom -D /dev/ttyUSB0 -b 1500000
```
