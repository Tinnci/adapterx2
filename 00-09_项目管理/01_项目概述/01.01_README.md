# AdapterX2 - USB-C UART 调试转接板项目

## 项目概述

本项目旨在设计两款专用的 USB Type-C 转 UART 调试转接板，分别针对：

1. **Samsung S9 系列** (1.8V 电平，MUIC 激活机制)
2. **Rockchip/PineNote 系列** (3.3V 电平，标准 DAM 机制)

## ⚠️ 重要警告

**这两款转接板不可互换使用！**

- 使用 3.3V 电平的转接板连接三星设备会**立即烧毁**手机主板
- 使用 1.8V 电平的转接板连接 Rockchip 设备可能导致信号无法识别
